Website Pemesanan Haji dan Umroh

Website pemesanan haji dan umroh ini dirancang untuk memudahkan jamaah dalam melakukan pemesanan paket perjalanan ibadah haji dan umroh secara online. Sistem ini menggunakan pembayaran manual melalui transfer bank untuk memastikan keamanan dan kemudahan dalam proses transaksi.

Fitur Utama:

  • Pendaftaran dan login pengguna
  • Pemilihan paket haji/umroh dengan detail lengkap
  • Formulir pendaftaran jamaah
  • Sistem pembayaran manual via transfer bank
  • Konfirmasi pembayaran oleh admin
  • Dashboard untuk tracking status pemesanan
  • Notifikasi via email dan WhatsApp

Teknologi yang Digunakan:

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP dengan framework Laravel
  • Database: MySQL
  • Pembayaran: Transfer bank manual
  • Notifikasi: WhatsApp API dan email SMTP

Keunggulan Sistem:

  • Interface yang user-friendly dan responsif
  • Proses pemesanan yang terstruktur dan aman
  • Integrasi notifikasi real-time
  • Manajemen data jamaah yang terorganisir
  • Laporan dan statistik untuk admin

Sistem ini telah berhasil diimplementasikan dan digunakan oleh berbagai travel umroh untuk memudahkan proses pemesanan jamaah. Dengan pembayaran manual, memberikan kepercayaan lebih kepada pengguna dalam melakukan transaksi.
