E-Umroh & Haji Management System

Sistem Informasi Manajemen & Reservasi Perjalanan Ibadah Terpadu.

Platform Digital Layanan Haji & Umroh

Platform ini dirancang khusus untuk memodernisasi proses bisnis biro perjalanan haji dan umroh. Fokus utama sistem adalah memberikan kemudahan bagi calon jamaah dalam memilih paket ibadah secara transparan, sekaligus menyediakan alat manajemen yang kuat bagi admin travel untuk mengelola manifes keberangkatan, dokumen jamaah, dan verifikasi keuangan dalam satu dasbor terintegrasi.

Fitur Unggulan:

  • Dynamic Package Catalog: Informasi detail paket meliputi jadwal penerbangan, hotel (bintang), rute perjalanan, dan sisa kuota kursi secara real-time.
  • Digital Pilgrim Form: Penginputan data identitas jamaah (Paspor, NIK, Vaksin) secara mandiri untuk akurasi manifes.
  • Manual Payment Validation: Alur upload bukti transfer bank yang dilengkapi dengan sistem notifikasi WhatsApp untuk konfirmasi instan oleh admin.
  • Document Tracking: Status pengurusan dokumen (Visa, Paspor, Manifes) yang dapat dipantau langsung oleh jamaah melalui dasbor pribadi.
  • Comprehensive Admin Dashboard: Manajemen inventory paket, laporan keuangan harian, serta ekspor data jamaah untuk kebutuhan manifes maskapai/hotel.

Arsitektur Teknologi:

  • Backend: Laravel PHP Framework (Keamanan data transaksional & pengolahan file PDF).
  • Database: MySQL (Optimasi relasi tabel jamaah, paket, dan transaksi).
  • Frontend: Clean UI menggunakan Tailwind CSS agar nyaman diakses oleh jamaah lansia sekalipun.
  • Notification: Integrasi WhatsApp Gateway & Email SMTP untuk update status pesanan.

Kontribusi Saya:

  • Membangun logika backend untuk sistem reservasi yang mampu mengunci kuota (booking seat) saat jamaah mendaftar.
  • Merancang skema database MySQL yang kompleks untuk menangani berbagai kategori paket (Bronze, Silver, Gold).
  • Implementasi fitur keamanan untuk proteksi dokumen sensitif jamaah (upload scan paspor/identitas).
  • Membuat fitur rekapitulasi data otomatis (Export Excel/PDF) untuk mempermudah operasional tim lapangan di tanah suci.
